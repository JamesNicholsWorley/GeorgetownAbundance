Gray dots = zone-ineligible under selected policy. 0 = MU/RA/CG zones only · 1 = + metro-adjacent R-zones (McDuffie) · 2 = + all R-zones districtwide (Lewis George)
Market
Market Rent 1.00×
Scales all submarket rents. City avg $3.65/SF at 1.00×. New Class A construction often 10–15% above existing stock.
Exit Cap Rate 5.00%
Investor required yield benchmark. CBRE Class A DC: 4.74% Q1 2025. YoC must exceed this by the spread below.
YoC Spread over Exit Cap 125 bps
Required excess return above exit cap rate. 125 bps = standard multifamily underwriting hurdle. Hurdle = exit cap + spread.
Inclusionary Zoning
IZ Set-aside — Wood Frame 11.0%
Affordable % for Type V (≤4 stories). DC baseline 10–12.5%; Lewis George proposes 15%. Applies to projects ≥10 units.
IZ Set-aside — Concrete/Steel 8.3%
Affordable % for Type I-mid and I (5+ stories). DC OP confirmed 8.0–8.33%; Lewis George proposes 13%.
Entitlement
Entitlement Speedup 0%
Reduction in entitlement timeline. McDuffie proposes 50% cut to review months. Affects carry cost via shorter debt draw period.
TOPA Friction 0 mo
Extra months added to entitlement for tenant opportunity-to-purchase process. Lewis George: +4 months.
Advanced Parameters
Cap Rates
Going-in Cap Rate 4.75%
Stabilized value = NOI ÷ this rate. CBRE DC Class A 4.74% Q1 2025. Lower = higher implied building value.
TOPA Exit Cap Premium 0 bps
Risk premium added to exit cap for TOPA encumbered assets. Lewis George: +30 bps (0.003). Raises YoC hurdle.
Construction Costs
Type V Hard Cost (≤4 stories) $280/SF
Wood-frame construction. RLB Quarterly Cost Report Q4 2025 DC. Includes structural, MEP, finishes.
Type I-mid Hard Cost (5–8 stories) $350/SF
Podium or wrap construction. Higher cost than wood-frame due to concrete podium and fire separation requirements.
Type I Hard Cost (9+ stories) $420/SF
Concrete or structural steel high-rise. Cumming Group DC 2026. Includes curtain wall, mechanical, vertical transport.
Cost Structure
Soft Cost 25.0%
Architecture, engineering, permits, legal — as % of hard cost. Higher in DC than national average due to entitlement complexity.
Contingency 10.0%
Hard cost buffer for unforeseen conditions. Applied as a multiplier on hard cost before soft cost is calculated.
Developer Fee 3.5%
Overhead and profit for developer, applied to hard + soft + project costs.
Financing
Construction Loan Rate 7.50%
Interest rate on construction debt. Approximate 2026 floating rate for ground-up multifamily. Affects carry cost.
Loan-to-Cost Ratio 65%
Fraction of hard + soft costs financed by debt. Remaining 35% is equity. Higher LTC = more carry cost risk.
Entitlement Timelines
Entitlement — By-Right 3 mo
Months for by-right administrative review (standard zoning compliance). Applies to most residential projects.
Entitlement — ZC/PUD 15 mo
Months for Zoning Commission map amendment or planned unit development. Applies when has_pud flag is set on the parcel.
Operations
Vacancy Rate 6.0%
Stabilized occupancy loss assumption. Applied to gross revenue to get effective gross income (EGI).
OpEx per Unit / Year $8,500
Property management, insurance, taxes, maintenance. IREM/NAA DC metro 2025 benchmark.
Avg Unit Size 1,000 SF
Average gross SF per residential unit. Affects unit count from GFA, IZ threshold, and per-unit opex.
Feasibility Thresholds
Min Return on Cost 6.5%
Untrended return threshold: stabilized value / TDC − 1. Parallel test to YoC; catches cases where NOI passes YoC but value creation is thin.
DC-Specific Fees
DC Water SAF (market unit) $22,580
⚠ System Availability Fee per market-rate unit. Figure not independently verified against dcwater.com FY2026 schedule. Use with caution.
DOB Permit Fee 2.0%
DC DOB permit fee as % of construction value, plus a 10% surcharge (fixed). Source: DC DOB fee schedule.
